# Dead Link

A static two-player text survival game for GitHub Pages.

## How to play

1. Open `index.html`.
2. The host enters a survivor name and copies the invite link.
3. The receiver opens that link, enters their name, and starts the game.
4. Players take turns choosing how to survive the zombie apocalypse.
5. Use **Copy Save Link** during the game if you want to reopen or share the exact current state.

## Publishing on GitHub Pages

Push these files to a GitHub repository, then enable Pages from the repository settings. No build step or backend is required.

Because this is a static game, the invite and save links store game data in the URL instead of using a server database.
